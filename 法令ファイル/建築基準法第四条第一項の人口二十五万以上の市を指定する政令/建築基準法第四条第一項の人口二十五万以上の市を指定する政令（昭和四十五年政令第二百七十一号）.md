# 建築基準法第四条第一項の人口二十五万以上の市を指定する政令（昭和四十五年政令第二百七十一号）
建築基準法第四条第一項の政令で指定する人口二十五万以上の市は、次の表の市の欄に掲げるとおりとする。
# 附　則
この政令は、昭和四十五年十月一日から施行する。
